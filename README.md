# 🚀 TechStore

A modern and responsive gadget store landing page built using **HTML5**, **Bootstrap 5**, and **Bootstrap Icons**.

## 🌐 Live Demo

Visit the live website:

**https://moyazmandal.github.io/TechStor/**

## 📸 Features

* Responsive Navigation Bar
* Hero Section with Call-to-Action
* Product Showcase Cards
* About Section
* Services Section
* Contact Form
* Modern Footer
* Mobile-Friendly Design
* Bootstrap 5 Grid System

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5.3
* Bootstrap Icons
* Google Fonts (Roboto)
